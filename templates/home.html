THREAT BASE

What is Threat Base?

Threat Base is an Threat Intelligence Sharing Platfrom where people can search for threat data information or can add threat data information themselves.

What is Threat Data?

Threat data is an amalgamation of malicious Hashes, IP addresses, URLs etc. It consists detected malicious agents which is used in previous Cyber Attacks. This Threat Data is used to prevent future Cyber Attacks.

What is the purpose of this Website?

By creating a website which allows people to contribute and search Threat Data, I aimed to improve Cyber Security of its users.
